## 웹 애플리케이션을 만드는 순서

1. 구상
2. 기획(계획): 무엇을 만들 것인가, 어떤 기능이 필요한가 등등
3. 디자인: 웹사이트를 아름답게, 사용자 편의성
4. 개발: 서비스가 실제로 동작하도록 함
5. 테스트: 버그나 놓친 부분 찾기

* 구상~테스트 과정을 계속 반복: Start Small
* 가장 중요한 것은 완성
* 복잡도: 기능을 추가할수록 복잡도는 지수적으로 증가함

### 기획

* UI 모델링: 직접 제작하기 전에 사이트 화면을 그려보기
  * User Interface: 사용자가 시스템을 제어하는 조작장치 (화면에 보이는 버튼 등)
  * 모델링: 사용자 입장에서 동작하는 기능을 가상으로 만들어보는 과정

* 모델링 도구: 손그림, Pencil, balsamiq, PowerPointㄴ

## 인터넷과 웹의 역사

* 인터넷 ⊃ 웹, FTP, 이메일
* 웹: 인터넷 위에 돌아가는 애플리케이션
* FTP: 파일을 전송할 때 사용하는 기술
* 이메일: 메세지를 주고받는 기술

 #### 인터넷

* 전세계의 컴퓨터들이 파일을 주고받을 수 있는 세계에서 가장 거대한 네트워크

#### 웹

* 전세계에 깔린 네트워크(인터넷)를 통해 HTML으로 만들어진 웹페이지를 컴퓨터끼리 주고받을 수 있게 해줌
* 팀 버너스리 개발

* 정보를 제공하는 웹서버 + 정보를 받는 웹브라우저

* 웹페이지의 정보를 담고 있는 HTML, 정보를 주고받는 통신규약인 HTTP

<br>

## 서버와 클라이언트

* <b>클라이언트</b>: 웹브라우저가 설치된 컴퓨터
* <b>서버</b>: 웹서버가 설치된 컴퓨터

* 클라이언트에서 웹브라우저에 주소를 침 -> 정보를 담고 있는 웹페이지를 서버 컴퓨터에 요청 -> 웹서버가 요청을 받아 하드에 저장된 웹페이지 문서를 읽어서 클라이언트 컴퓨터에 응답
* 즉 <b>클라이언트는 요청, 서버는 응답</b>

* bitnami, APMSetup 등은 Apache, php, MySQL 등이 포함된 통합환경
* localhost/index.html
  * localhost: 웹서버를 가리키는 주소
  * /index.html: 브라우저가 웹서버에게 Document Root의 index.html이라는 페이지를 요청

<br>

## HTML

* Hypertext Markup Language
  * Hypertext: 문서와 문서가 링크로 연결됨
    * <b>링크</b>로 문서와 문서가 그물망처럼 연결되어 하나의 거대한 정보 덩어리를 만든 것이 웹
    * 링크는 HTML의 본질, HTML의 본질은 웹의 본질
  * Markup Language: HTML은 사람도 웹 브라우저도 알아들을 수 있는 언어

#### Semantic Web

* 정보의 의미를 잘 드러내는 웹 (정보를 의미론적으로 표현하는 태그를 사용한 웹)
* \<nav>: 웹사이트를 탐색하는데 사용하는 정보임을 나타내는 태그

* \<header>: 문서의 간판임을 나타내는 태그

* \<article>: 문서의 본문임을 나타내는 태그

#### index.html

* 암묵적으로 웹사이트의 첫 페이지임을 의미
* / 뒤에 아무것도 붙이지 않으면 우선적으로 index.html을 찾아서 표시함

<br>

## CSS

* Cascading Style System: HTML 디자인을 위한 언어

* HTML은 정보에 집중, CSS는 디자인
* h1 {color:red}
  * 선택자(selector) : 지정할 태그 (h1)
  * 서술(description) : 선택자에게 줄 효과 ({color:red}
* \<link>: 외부에 분리된 css 파일을 불러오는 태그

#### 박스모델

* content: 내용
* width, height: content의 너비와 높이
* border: 테두리
* padding: content와 border의 간격
* margin: 다른 박스와의 간격

#### float

* 여러 컨텐츠를 나란히 나열하는 기능
